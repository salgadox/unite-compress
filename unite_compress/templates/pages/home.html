Welcome to our file compression and course management system!

Here you can compress your files and organize them into courses.

Upload a file Create a course

How it works

  1. Create a Folder: Go to the Create Folder page and enter a name and a description for your folder.
  2. Upload a file: Go to the Upload File page and select the file you want to upload. You can choose to add it to an existing folder or create a new one.
  3. Compress the file: Go to the file detail page by clicking on its name in the file list and click on the "compress" button. You can choose to compress the file with the default "medium" compression level or click on the "Advanced options" button to choose from three different compression levels.

The platform supports the following mime types :

  • “application/pdf"
  • "images/*"
  • “video/*”
